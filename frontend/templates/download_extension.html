Installation Instructions:

  1. Download the extension zip file above
  2. Extract the zip file to a folder on your computer
  3. Open Chrome browser and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top right corner)
  5. Click "Load unpacked" button
  6. Select the extracted folder (chrome_extension)
  7. The Bharati AI extension icon should appear in your Chrome toolbar
  8. Click the extension icon and login with your dashboard credentials
  9. Go to Gmail and open any email - it will automatically classify and generate a draft reply!
